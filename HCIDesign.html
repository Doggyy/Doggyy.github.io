HW4-2 : HCI Design Posted by Yoon Minho on November 16, 2018

A. Requirements:

i. Describe the overall objective of the application (one paragraph)
  요즘 스마트폰 카메라 성능의 발전과 저장공간의 여유 덕분에 많은 사람들이 주위 풍경이나 음식 사진을 많이 찍는다. 또한 낮의 구름 사진이나 야경 사진을 찍는 경우들도 많다. 그러다보니 스마트폰에 많은 사진 파일들이 쌓이게 되었고, 각자 사진을 분류하고 정리하기도 하지만 굉장히 번거롭고 귀찮은 일이 되어버렸다. 그래서 정리를 포기하는 경우도 드물지 않게 있는데 이를 돕기 위해 사진들을 찍은 장소, 날짜, 찍은 시간 등등으로 자동으로 분류해준다.

ii. List major functional requirements (3 ~ 4)
  ㆍ 저장장치에 저장된 사진 파일들을 익스플로러 형태로 보여준다.
  ㆍ 날짜별로 자동 정리하여 볼 수 있다.
  ㆍ 장소 혹은 지역 별로 자동 정리하여 볼 수 있다.
  ㆍ 스스로만의 카테고리를 만들어 정리할 수도 있다.

iii. List major UI requirements (more than 4, total)
  1. Functional
    ㆍ 사진을 삭제하거나 수정하는 버튼
    ㆍ 날짜별로 정리할 지 지역별로 정리할 지 시간대 별로 정리할 지 선택하는 메뉴
    ㆍ 스스로만의 카테고리를 만들 수 있는 버튼
    ㆍ 사진 선택 시 크게 보여줌
  2. Non-functional
    ㆍ 너무 어둡거나 밝은 경우 사진을 보기 힘들 수 있으므로 약간 밝은 회색의 배경 사용
    ㆍ 사진의 분별성을 위해 여러 사진을 보여줄 때 너무 작게 보이지 않도록 함
    ㆍ 적당한 사진간의 간격으로 가시성을 높임

iv. List other considerations (any)
  안드로이드 OS를 사용하는 안드로이드폰을 타겟. 타겟은 사진을 많이 찍는 20대~30대. 요즘엔 기능만 좋은 게 아닌 감성적으로도 만족할 수 있는 앱을 많이 찾으므로 UI와 앱 디자인에 신경을 많이 써야함.

B. User analysis

  i. Conduct user interviews (2-3 people) from your target user group. Recode a short video of the interview (hide the interviewee).
    링크

  ii. List the major direct user requirements? (3 ~ 4)
    ㆍ 다운받은 사진들은 따로 분류해줬으면 좋겠다.
    ㆍ 지역별로 분류하는 경우 국가, 도, 시 등으로 나누는 척도도 선택할 수 있음 좋겠다.
    ㆍ 날짜별로 분류하는 경우 년, 월, 일 등으로 나누는 척도도 선택할 수 있음 좋겠다.

  iii. List the major cognitive/ergonomic requirements? (more than 2)
    ㆍ 보통 메뉴가 오른쪽에 있으니 분류를 선택하는 버튼은 우측 상단에 있었으면 좋을 것 같다.
    ㆍ 분류 방법 선택을 한 뒤 화면 왼쪽 끝을 오른쪽으로 스크롤하면 다른 카테고리도 보여주면 좋을 것 같다.
    ㆍ 사진을 확대해서 볼 때 스와이프로 다른 사진들로 넘길 수 있으면 좋을 것 같다.

  iv. Construct 2-3 usage scenarios. Use Powerpoint or any drawing software and annotate important issues.

  ㆍ 메인 화면


  앱을 처음 켰을 때 모습이다. 처음에는 폴더 별로 분류하여 보여주며 폴더를 선택하여 원하는 사진들을 볼 수 있다. 우측 상단에는 분류 방법을 선택할 수 있는 버튼이 있다. 분류 버튼을 누르면 분류 종류가 나타나며, 이 중 하나를 선택하면 원하는 분류 방법으로 분류하여 보여주게 된다.

  ㆍ 분류 선택시


  분류를 선택하게 되면 그에 맞게 사진들을 분류해준다. 폴더 형태로 정리되어 있어 파일 익스플로러 사용하듯이 사용하면 된다. 날짜의 경우 년도 선택 -> 월 선택 -> 일 선택 순으로 폴더를 선택하면 되고, 지역의 경우 국가 선택 -> 도 선택 -> 시 선택 순으로 선택하면 된다. 이때 화면 왼쪽 끝부분을 스와이프 해주면 다른 카테고리를 보여주어 빠르게 다른 카테고리로 넘어갈 수 있다.

  ㆍ 사진 선택시


  사진을 확대해서 보여주며 양쪽으로 스와이프 하여 다른 사진으로 넘어갈 수 있다. 사진을 꾹 눌러 사진을 수정하거나 삭제하는 것도 가능하다.

v. Construct a rough interaction model (e.g. like a GOMS model) for 2-3 major tasks.
1. 처음 사진 탐색
해당 폴더 선택 -> 세부적인 폴더 선택 -> 사진 선택
2. 사진 분류하여 탐색
우측 상단 분류 방법 터치 -> 원하는 분류 방법 선택 -> 분류된 폴더 중 원하는 폴더 선택 -> 세부적인 폴더 선택 -> 사진 선택
3. 사진 넘기기
화면 터치 -> 원하는 방향으로 스와이프
4. 다른 카테고리 선택
화면 왼쪽 끝 터치 -> 오른쪽으로 스크롤
